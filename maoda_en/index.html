Research Statement

My research focuses on developing human-centered, intelligent data analysis workflows, with a special emphasis on visualization techniques that make complex data accessible and actionable for both experts and the general public. Visualization techniques leverage the vast bandwidth of visual channels to express large-scale data, aiding users in understanding, exploring, and discovering knowledge. Moreover, with interactive functions and large-scale data management technologies, it's possible to perform multi-level, demand-based exploration of financial data, for instance, visual analysis methods can support the detection of risky behaviors in blockchain transactions. During my PhD, I proposed a series of deep learning-supported data analysis algorithms, covering large-scale data management for real-time analysis, intelligent generation of data visualizations, generation and parsing of natural language content, and user-oriented data visualization natural interactions. Representative research includes large-scale data management for interactive data visualization, natural language content construction for data visualization based on deep learning, automatic interaction for static visualizations based on force metaphor, and easy-to-understand interaction methods combining direct manipulation and natural language. This intelligent data workflow covers new types of interactions from data to users, aiming to support users in efficient data analysis and understanding. Combining the data analysis needs related to finance with my knowledge background and technical accumulation, I plan to conduct a series of research in the following directions:
  1. Intelligent Financial Data Analysis Platform: Existing financial data sources are diverse (from government, enterprises, third-party institutions, and media, etc.), structurally complex (tables, networks, texts, etc.), and lack a unified standard. A standardized data platform is the basis for many data-driven research results and has significant academic and social impact. My proposed research points include financial data collection, financial data management, and intelligent interactive analysis. I plan to combine large language models and other deep learning algorithms to create automated financial data processing algorithms for crawling and organizing, adaptive algorithms for large-scale high-dimensional data management, and automated systems for financial data visual analysis and automatic financial knowledge discovery.
  2. Intelligent Interaction-Supported Exploration of Financial Phenomena: Existing financial technology fields often use statistics, regression, and other methods to calculate indices and factors from large-scale data. This process, essentially a data abstraction process, inevitably loses a lot of data information. I plan to develop an interactive large-scale financial data visualization analysis system. For different phenomena and tasks, using network structure visualization, text visualization, high-dimensional data visualization, and new visualization methods tailored for financial problems, I aim to build a big data system that goes from global to detail, explores as needed, and provides real-time feedback, thereby further constructing a closed loop of financial data-financial modeling-knowledge discovery with humans in the loop.
  3. Data Analysis Tools for Financial Experts: Current financial technology-related research requires a lot of mathematical and computational methods. Mathematical principles and formulas need to be assisted by program codes for analysis. Existing data analysis software and programming languages are not very user-friendly and are not the best practice. I plan to develop a series of interactive programming tools for practical financial analysis needs, using natural language, graphical interfaces, and other interaction methods to lower the threshold of use and support integration with existing data analysis software.
